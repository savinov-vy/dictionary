1. Аннотации находится в пакете org.junit.jupiter.api
2. Класс Assertions в пакете org.junit.jupiter.api.Assertions
3. @Before, @After -> @BeforeEach, AfterEach
4. @BeforeClass, @AfterClass -> @BeforeAll, @AfterAll
5. @Ignore -> @Disabled
6. @RunWith -> @ExtendWith
7. Аннотация @Test не принимает аргументы expected и timeout ->
-> вместо них методы класса Assertions: assertThrows(), assertTimeout
