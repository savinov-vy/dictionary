git
вывести команды git

-----------

git help <команда>
# git help reset
вывести описание о конкретной команде

-----------

git init

создать скрытую папку .git в директории проекта
содержит метаинфомацию о проекте

-----------
удалить скрытую папку .git
rm -rf my_project
-----------

посмотреть есть ли что сохранить в репозиторий
git status

------------
добавить все файлы в staging area и приготовть для сохранения
git add .

------------
создать коммит с сообщением

git commit -m "Initial commit" 

------------
история изменений содержащее хеш изменения

git log
------------
вывести все ветки 
git branch

------------

создать новую ветку

git branch bugFix#100

------------

перейти на новую ветку
git checkout bugFix#100

------------
создать и сразу перейти на новую ветку
-b  - branch создать ветку

git checkout -b bugFix#100

------------
Удалить ветку. Нельзя удалить активную ветку

git branch -d bugFix#101
------------
Мерджить ветку в мастер
Перейти в ветку мастер
ввести команду
git merge newBranch
где newBranch название ветки изменения которой хотим влить
