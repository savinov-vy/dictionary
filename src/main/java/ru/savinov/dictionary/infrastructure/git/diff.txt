
git diff
показывает разницу между текущим неотслеживаемым состоянием репозитория
и последним снимком репозитория. Показывает только изменения в файлах. Новые файлы не показывает

--------
git diff --staged
показывает разницу между текущим отслеживаемым состоянием репозитория и последним снимком репозитория
показывает добавления в новых файлах, но файлы должны быть отслеживаемые добавленные в git storage

противоположность в некотором смысле на git diff
--------

git diff COMMIT_ID 

показывает разницу между текущим состоянием репозитория и указанным снимком репозитория
