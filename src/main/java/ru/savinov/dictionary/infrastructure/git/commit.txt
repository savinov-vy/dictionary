-----------------
git commit -a -m "сообщение коммита"
тоже что и последовательное выполнение 
git add .
git commit -m "сообщение коммита"

отличие в том что флаг -a не добавляет untrached файлы. 
Добавляет только модифицированные (modified) файлы
-----------------

git add .
git commit --amend -m "Новое сообщение коммита"

выйти из текстового редактора

дополняет послдений коммит, добавляя в него "Свежие изменения".
сливает 2 последних коммита с последними изменениями и "Новым сообщением коммита"
