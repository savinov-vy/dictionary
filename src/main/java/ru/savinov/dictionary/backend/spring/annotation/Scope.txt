Аннотация @Scope("singleton") используется для укзания спрингу scope объекта т.е. будет ли объект создаваться сразу
при инициализации IoC контейнера и иметь единственную реализацию. Или бин будет создаваться при обращении к контексту
за необходимым бином и для каждого запроса (инъекции зависимости) будет создаваться свой бин.

@Component
@Scope("prototype")
public class Dog implements Pet{
...................
}

подробнее о жизненных циклах бина можно почитать beans lifeCycle
