Процесс внедрения зависимостей при использовании @Autowired

- Сканирование пакета, поиск классов с аннотацией @Component
- При наличии аннотации @Autowired начинается поиск подходящего по типу бина
- Если находится 1 подходящий бин происходит внедрение. В ином случае выбрасывается исключение

Начиная со спринга 4.3 если в классе всего один конструктор то аннотацию Autowired ставить не обязательно. Внедрение
зависимостей всё равно произойдет.
