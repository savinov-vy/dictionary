Класс теста должен начинатся с названия тестируемого класса заканчиваться на словом Test
ClassNameTest

--------------------------------------------------------------------------------------------------------

Метод теста для unit 5 может быть обязательно public void или дефолтным т.е. просто void (private не может
иначе тест не взлетит. Метод дожен начинаться на слово test.
Тестироваться в каждом тесте должна только одна проверка.

void testNameUseMethod_example1()


Каждый тест должен быть помечен аннотацией тест.


--------------------------------------------------------------------------------------------------------

Метод помеченный аннотацией @BeforeEach называется init() или setUp() выполняется каждый раз перед выполнением каждого
метода помеченного аннотацией тест

---------------------------------------------------------------------------------------------------------

Метод помеченный аннотацией @AfterEach называется destroy() или tearDown()
