не точно:
разница использования метода session.find(Object.class, id) и session.get(Object.class, id)
заключается в том что метод get должен доставать объект всегда из базы данных
а метод find () сначала ищет в контексте хранения, потом если не находит начинает искать в базе
при повторном использовании должен уже находить в контексте хранения (своеобразная кеш память).

get() в нутри себя делает load()

session.clear() чистит кеш нутри контекста хранения

session.detach() выкидвает подгруженный объект из контекста хранения
